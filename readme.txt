SuperBuzz - Increase engagement and drive more repeat traffic to your WordPress site with push notifications.

== Description ==

[SuperBuzz](https://superbuzz.io) is an easy way to increase user engagement. Use SuperBuzz to send visitors targeted push notifications so they keep coming back. It takes just a few minutes to install.

SuperBuzz has been rigorously tested to handle even the highest volume use cases with reliability and ease of use.

After setup, your visitors opt-in to receive push notifications when you publish a new post. Visitors receive these notifications even after they’ve left your website, thus driving re-engagement.

= Features =

* **Supports Chrome** (Desktop) and **Firefox** (Desktop) on HTTPS sites.

* **Automatic Notifications** - Send notifications to followers every time you publish a new post. Or set up a reminder that gets automatically sent to them if they haven’t visited for a few days.

* **Targeting Segments** - Send notifications to specific visitors based on language, number of times they’ve visited your blog, or even set up your own user attributes that you can target.

## SuperBuzz 3rd Party as external service

The plugin uses 3rd party external service developed by SuperBuzz in order to communicate with SuperBuzz servers:

1. The push notification service worker file https://app.superbuzz.io/sdk/SuperBuzzSDKWorker.js:

SuperBuzz service worker library for browser push notifications, this file is required by browsers to register push notification service worker.

2. The SuperBuzz js library file https://app.superbuzz.io/SuperBuzzSDK.lib.js:

SuperBuzz SDK library file to manage push notifications flow from SuperBuzz server with authenticated website owner credentials
and also responsible for sending website visitors events to SuperBuzz AI server.

== Installation ==

1. Install SuperBuzz from the WordPress.org plugin directory or by uploading the SuperBuzz plugin folder to your wp-content/plugins directory.
2. Activate the SuperBuzz plugin from your WordPress settings dashboard.
3. Follow the instructions on the Setup page.

== Frequently Asked Questions ==

#### How do I enable push notifications for my site? ####

To enable push notifications for your site using SuperBuzz, follow these steps:

1. Install and activate the SuperBuzz plugin.
2. Go to the SuperBuzz settings page in your WordPress dashboard.
3. Click on the "Setup" tab and follow the instructions to configure your push notification settings.
4. Save the changes, and push notifications will be enabled for your site.


#### Are push notifications supported on mobile devices? ####

Currently, SuperBuzz supports push notifications on desktop browsers such as Chrome and Firefox. Mobile support is not available at this time.


#### Is SuperBuzz compatible with other plugins? ####

SuperBuzz is designed to work well with most plugins that follow WordPress coding standards. If you encounter any compatibility issues, please reach out to our support team for assistance.


#### How can I get support for SuperBuzz? ####
